create a new repository on the command line

echo "# Web-Developer-Portfolio" >> README.md
git init
git add README.md
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/nuruzzaman-pfy/Web-Developer-Portfolio.git
git push -u origin main

…or push an existing repository from the command line

git remote add origin https://github.com/nuruzzaman-pfy/Web-Developer-Portfolio.git
git branch -M main
git push -u origin main
